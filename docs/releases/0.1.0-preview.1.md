# MoonDesk 0.1.0 Preview 1

This is a preview build of the MoonBit-native MoonDesk desktop application,
packaged with Lepusa for Apple silicon Macs.

## Highlights

- Refreshed Rabbita desktop interface and MoonFlow composition canvas.
- Self-contained Lepusa runtime, MoonDesk sidecar, production UI, and app icon.
- Portable bundle-relative service and UI paths.
- User data stored under Lepusa's macOS application-data directory rather than
  inside the signed application bundle.
- Reliable last-window and signal shutdown of the supervised localhost service.
- Self-contained generated install-smoke script.

## Compatibility

- Architecture: macOS arm64 (Apple silicon)
- Minimum macOS version: 11.0
- WebView: system WKWebView

## Preview signing notice

This preview is ad-hoc signed and is not Apple-notarized. Gatekeeper may require
Control-clicking the application and choosing **Open** on first launch. Do not
treat this preview as the final public distribution build.

## Verification

- MoonDesk: 204 MoonBit tests and 2 JavaScript i18n tests passed.
- Lepusa: 542 MoonBit tests passed.
- Strict Lepusa verification, DMG verification, install smoke, packaged UI
  navigation, post-launch signature verification, and supervised-service
  shutdown passed.
- Both native executables declare macOS 11.0 as their deployment target.
