# MoonDesk 0.1.0 Preview 2

This preview refreshes the MoonBit-native MoonDesk desktop application for
Apple silicon Macs with the latest MoonCode failure-state fix.

## Changes since Preview 1

- Release a failed new-chat draft immediately when MoonCode session creation
  fails, so the composer can recover instead of remaining gated.
- Show the failed draft as an assistant error row with the provider error.
- Add regression coverage for failed MoonCode session creation.
- Rebuild the production Rabbita UI with the fix.
- Include both the app and an `/Applications` shortcut in the DMG.

## Compatibility

- Architecture: macOS arm64 (Apple silicon)
- Minimum macOS version: 11.0
- WebView: system WKWebView

## Preview signing notice

This preview is ad-hoc signed and is not Apple-notarized. Gatekeeper may require
Control-clicking the application and choosing **Open** on first launch.

## Verification

- MoonDesk: 205 MoonBit tests and 2 JavaScript i18n tests passed.
- Native checks, MoonBit formatting, and generated interfaces passed.
- Strict Lepusa verification, DMG integrity, app signature, packaged UI asset,
  minimum macOS version, and `/Applications` shortcut checks passed.
