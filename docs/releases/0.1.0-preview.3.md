# MoonDesk 0.1.0 Preview 3

This preview corrects the macOS application identity and installer layout.

## Changes since Preview 2

- Rename the installed application from `moondesk-lepusa.app` to
  `MoonDesk.app`.
- Set the bundle name and display name to `MoonDesk`; Lepusa remains an
  internal runtime implementation detail.
- Generate the DMG with `MoonDesk.app` and the standard `/Applications`
  shortcut directly from the release command.
- Keep the installer filename `moondesk-0.1.0-macos-arm64.dmg`.

## Compatibility

- Architecture: macOS arm64 (Apple silicon)
- Minimum macOS version: 11.0
- WebView: system WKWebView

## Preview signing notice

This preview is ad-hoc signed and is not Apple-notarized. Gatekeeper may require
Control-clicking the application and choosing **Open** on first launch.

## Verification

- MoonDesk: 205 MoonBit tests and 2 JavaScript i18n tests passed.
- Native checks, MoonBit formatting, and generated interfaces passed.
- Strict Lepusa verification, DMG integrity, app signature, bundle/display
  names, packaged UI assets, and `/Applications` shortcut checks passed.
