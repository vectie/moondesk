## 1. External model capsules

MoonClaw may send `gpt-5.6-sol` an isolated capsule containing:

- the immutable work request, original acceptance criteria, declared product
  contract, and public contract guidance;
- the original lunar-robot goal;
- only the declared accepted predecessor artifacts needed by that work item;
- on bounded retries, the prior partial draft and structural rejection reasons.

The capsule excludes credentials, tokens, private repository contents,
absolute host paths, unrelated books, unrelated workspace files, and authority
to create external or physical effects.

## 2. Public research requests

Only the observe-only MoonWiki evidence item may use public Web tools. It may
send search queries derived from the approved goal and fetch selected public
URLs. The wrapper records each successful search and fetch event. MoonBook's
native attestor rejects source locators that do not equal a successfully
fetched URL.

No other product request receives public Web tools. No publication, user
communication, purchase, physical command, launch, or deployment is allowed.

## Approval semantics

Approval is recorded independently for the bounded model capsules and the
public-research requests. It does not authorize broader repository export, secrets,
external publication, physical work, or a future run. MoonGate will issue a
new expiring digital-only envelope after approval; the expired r1 envelope is
never extended or reused.
